<objective>
Rewrite the design-system foundation of foz-foco-real so every downstream plan can compose UI using only Tailwind utilities backed by the Stitch M3 palette and the Newsreader + Plus Jakarta Sans type system.

Purpose: The prior commit (b4a69e3) was rejected because it only swapped tokens without removing the old visual system. This plan deletes the Sora / Bebas / DM-* fonts, deletes the legacy editorial CSS classes (.hero-section, .article-card, .main-grid, .footer, .business-grid, .nav-pill, etc.), installs the exact Stitch palette from RESEARCH.md §1.5, and recalibrates the shadcn HSL dark-mode tokens. Output: New tailwind.config.ts, rewritten globals.css (much smaller), updated app/layout.tsx with next/font loaders, and minimal patches on the three shared public pages to replace deleted class names. No component rewrites yet — those belong to plans 01-02, 01-03, 01-04.
</objective>
